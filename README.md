### Getting Started

1. Clone the repository
```bash
git clone https://github.com/ahump20/blaze-intelligence-official.git
cd blaze-intelligence-official
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm start
```

4. Build for production
```bash
npm run build
```
